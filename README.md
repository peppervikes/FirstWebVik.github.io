# Simple Express Server Backend

## Overview
This project represents a simple backend service built with **Node.js** and **Express**. It was designed as an introductory foray into server-side programming and routing. It serves static assets and basic API responses for an application.

## 🚀 Key Learning Outcomes
- Understanding server setup using Node.js and the Express framework.
- Configuring entry points for backend web applications.
- Routing and handling HTTP requests.
- Serving assets for standard web applications.

## 💻 Tech Stack
- **Node.js**
- **Express.js**

## 🛠️ Usage
To run this application locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/peppervikes/FirstWebVik.github.io.git
   ```

2. Navigate into the application directory:
   ```bash
   cd FirstWebVik.github.io/dog-services
   ```
   *(Note: ensure that you install dependencies if package.json has updates by running `npm install`)*

3. Start the application:
   ```bash
   node app.js
   ```

4. Open your browser and navigate to `http://localhost:3000/`.
